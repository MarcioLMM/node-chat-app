# Projeto Web 
- Projeto realizado para a disciplina tecnologias para desenvolvimento web, professor João Cunha.

# Tecnologias utilizadas
- HTML5
- CSS(Materialize)
- JQuery
- Axios(biblioteca para fazer requisições para o back de maneira simples)

- Node
- Express(Framework MVC responsável pelo backend)
- Socket-io(Framework responsável pela comunicação real time do projeto)
- Sequelize(Framework que faz a conexão com o banco de dados, operações CRUD)
- MySQL(Banco de dados que é salvo as informações sobre usuários e mensagens)
- Redis(Banco de dados in memory, extremamente rápido, responsável por armazenar informações sobre quem está online)

# Projeto
- Foi criado uma web chat para usuários possam se cadastrar e conversar real-time com outras pessoas conectadas
- É possivel realizar operações CRUD nos usuários
- Site é responsivo
